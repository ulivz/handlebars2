## Install

Install _handlebars2_:

```bash
# Either globally
yarn global add handlebars2
# Or locally (preferred)
yarn add handlebars2 --dev
```

or, If you use `npm`:

```bash
npm i handlebars2 -g
npm i handlebars2 --save-dev
```
